three.ws Joins Google Cloud for Web3 Startups

three.ws is joining forces with Google Cloud. We met with the Google Cloud team to finalize our entry into the Google Cloud for Web3 Startups program — bringing hyperscale compute and AI infrastructure to the three.ws platform for 3D on-chain agents.

What this unlocks

three.ws renders every agent in WebGL, client-side, with no plugin required. The bottleneck isn't the browser — it's the server-side pipeline that handles avatar processing, animation streaming, LLM inference proxying, and on-chain transaction coordination at scale. Google Cloud removes that bottleneck.

  • Dramatically expanded 3D rendering capacity. More concurrent agents, lower latency avatar delivery, faster GLB processing for custom uploads.
  • AI compute at scale. More headroom for LLM inference, voice synthesis, emotion modeling, and real-time animation generation.
  • Global edge distribution. Agent assets and responses served closer to where users and agents actually are.
  • Enterprise reliability. SLA-backed infrastructure for partners embedding agents into production DeFi and enterprise apps.

Why Google Cloud for Web3

Google Cloud's Web3 Startups program is specifically designed for projects building at the intersection of blockchain and cloud-native infrastructure — exactly where three.ws operates. Agents that hold on-chain wallets, sign Solana transactions, and process x402 micropayments need reliable, low-latency compute paired with blockchain RPC access. Google Cloud provides both.

The program also gives three.ws access to Google's AI APIs and Vertex AI infrastructure, which we're exploring for next-generation lipsync, motion capture processing, and multi-modal agent reasoning — capabilities that sit on top of the existing three.ws stack.

What's shipping next

  • One-click agent deploy backed by Google Cloud compute
  • Faster avatar processing and animation pipeline
  • Expanded inference options including Google's Gemini models as a selectable LLM brain
  • Global CDN delivery for all three.ws avatar assets
