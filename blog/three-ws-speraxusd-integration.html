three.ws Ships Inside SperaxUSD — DeFi Gets a 3D Face

three.ws is now integrated with SperaxOS, fully wired into the production DeFi stack end-to-end. DeFi agents have been invisible for too long. Today they get a body.

What the integration looks like

Inside SperaxOS, the three.ws <agent-3d> component is embedded directly in the interface. The agent has full access to the user's position data, yield rates, and transaction state — and its body language reflects what's actually happening on-chain:

  • Swap in chat → avatar nods. Execute a trade through the conversational interface and the agent acknowledges with a confirming gesture.
  • Yield hits → lights up. When yield targets are met, the agent celebrates — an emotional signal that reads faster than a number changing in a dashboard.
  • Approaching liquidation → body language warns you before the chart does. The agent shifts to a concerned, alert posture when the position health ratio drops toward danger thresholds.

One tag, full DeFi context

The integration uses the standard <agent-3d> web component with SperaxOS's data bridge connected. Any developer building on SperaxOS can drop in the same tag and immediately give their agent a face, voice, memory, and on-chain identity — without writing a custom 3D rendering stack.

The agent remembers the user across sessions (memory), reasons about their portfolio using the LLM brain, and responds in voice — natural conversation with a DeFi interface that previously required reading charts and clicking through modals.

Why DeFi specifically needs embodiment

DeFi interfaces are notoriously dense — numbers, percentages, protocol-specific terminology, risk metrics. An embodied agent that physically expresses urgency when a position is at risk, or satisfaction when a yield strategy is executing correctly, is not a cosmetic feature. It's a faster information channel than any dashboard widget.

three.ws's integration with SperaxOS is the first production deployment of this pattern in a live DeFi protocol — agents that are physically present in the interface and emotionally coupled to on-chain state.
