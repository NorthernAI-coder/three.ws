02

Three doors, one artifact

Install

Pick whichever fits your stack. All three forms ship from the same bundle and render identically.

1 · Custom element

For HTML, MDX, Webflow, Framer. Drop the tag where you want the agent.

<script src="https://three.ws/embed/v1.js" async></script>
<three-d agent="8453:42"></three-d>

2 · Script-tag drop-in

Zero markup. The script auto-mounts itself in place — perfect for Squarespace, Substack, and no-code surfaces.

<script src="https://three.ws/embed/v1.js"
        data-agent="8453:42" async></script>

3 · JS API

For React, Vue, Svelte, or any framework that builds the DOM itself.

import 'https://three.ws/embed/v1.js';
Three.mount('#root', {
  agent: '8453:42',
  interactive: true,
});

4 · Direct GLB

Skip the resolver — point at any glTF/GLB URL. Useful for assets you already host.

<three-d src="https://you.com/model.glb"
         name="My agent"></three-d>
