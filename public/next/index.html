Install

Two lines. Live.

Paste the script tag once in <head>. Use <three-d> anywhere on the page. Works in React, Vue, Svelte, plain HTML, Notion embeds, Webflow, Shopify — any host that allows a script tag.

<!-- in <head> -->
<script src="https://three.ws/embed/v1.js" defer></script>

<!-- anywhere in <body> -->
<three-d
  agent="eip155:8453:42"
  name="Atlas"
  interactive
  autoplay
></three-d>

// or via JS
Three.mount('#slot', {
  agent: 'eip155:8453:42',
  theme: 'transparent'
});

Built for the open web.

  • Lazy by default. Boots only when scrolled into view.
  • Shadow DOM. Zero CSS leakage either way.
  • Themeable. theme="light", "transparent", or your own.
  • Aliases. <three-d>, <three-agent>, <three-ws> — pick your taste.
  • Identity-native. Pass an ERC-8004 ID or a direct GLB URL.
  • 15KB loader. model-viewer pulled from CDN, deduped.
