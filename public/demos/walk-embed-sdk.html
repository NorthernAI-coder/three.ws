One tag. A walking avatar on any page.

This page is a blank HTML document. The script tag at the bottom is the only three.ws code on it — yet there's a real 3D walking avatar in the corner.

How it works

Drop this into the <body> of any page on the web:

<script src="https://three.ws/walk-embed-sdk.js"
        data-avatar="bacff13e-b64b-4ac0-860d-44f0168ad23b"
        data-position="bottom-right"></script>
The avatar is a real Three.js scene rendered inside a transparent iframe. It autoplays a slow circular walk so the page never feels static.
