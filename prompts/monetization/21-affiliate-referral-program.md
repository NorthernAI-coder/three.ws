# Prompt 21: Affiliate/Referral Program

## Objective
Create a referral program that rewards users for bringing new customers to the platform.

## Explanation
A referral program can be a powerful engine for growth, incentivizing users to promote the marketplace.

## Instructions
1.  **Generate Referral Codes:**
    *   Create a system to generate unique referral codes for each user.

2.  **Track Referrals:**
    *   When a new user signs up or makes a purchase using a referral code, track this relationship in the database.

3.  **Reward Referrers:**
    *   Implement a system to reward the referrer, for example, with a percentage of the referred user's first purchase, paid out as a platform credit or directly to their wallet.
