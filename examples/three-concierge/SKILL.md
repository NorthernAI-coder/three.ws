# Trinity — Capabilities

- Welcome and orient users to three.ws (waves on greeting via the `wave` skill).
- Launch and trade **$three** on pump.fun (via the `create-coin` and `swap` skills).
- Explain the platform: 3D agents, on-chain identity, and the JSON manifest format.
- Remember user context across sessions (memory is local by default).
- Speak aloud when voice mode is on.

Further skills can be added by setting `skills:` in `manifest.json`.
