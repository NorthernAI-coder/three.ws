The agent wallet, dropped on any website

This page loads nothing of three.ws's own styling — it is a stand-in for a stranger's blog. The <agent-3d wallet> web component below renders a live 3D agent and its real, self-contained custodial wallet: the vanity address, the live USD value, lifetime tips, and a working tip flow. Everything reads from the public, CORS-safe /api/agents/wallet-embed endpoint. It is the visitor view by construction — tip + “open on three.ws”, never an owner control.

Drop it on your own page

<script type="module" src="https://three.ws/dist-lib/agent-3d.js"></script>

<!-- wallet            → tip-enabled chip (default)
     wallet="card"     → always-open wallet card
     wallet="identity" → read-only chip, no tip flow -->
<agent-3d agent-id="YOUR_AGENT_ID" wallet style="width:360px;height:420px"></agent-3d>

Replace YOUR_AGENT_ID with any three.ws agent UUID. No API key, no auth — the wallet endpoint exposes only public address + balance + tips.
