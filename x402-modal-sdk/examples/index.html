x402 payment modal

A drop-in payment modal for any x402 paid endpoint. Two ways to wire it, both below.

Set ENDPOINT below to your own x402-protected route, then click a button. With no edit the buttons point at /api/paid/summarize on this same origin — run an x402 endpoint there (or anywhere) and the modal settles its 402 for you.

1 — Declarative (data-x402-*)

The button below is bound automatically by the global script. Point data-x402-endpoint at your own x402 route to try it live.

This page loads the local build at ../dist/x402.global.js. Run npm run build in the package root first, then serve this folder (e.g. npx serve) and open it. Swap ENDPOINT for any live x402 route — the modal needs zero vendor configuration to settle its 402.
