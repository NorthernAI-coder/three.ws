API

Free endpoints

IP-rate-limited, generous, no key required. Every response carries asOf + source.

Paid endpoints

Metered per-call via x402 — USDC on Base (mainnet) or Base Sepolia (testnet). No account, no invoice, no subscription.

Paying with x402

Every paid route returns HTTP 402 with a signed PaymentRequirements challenge when called without payment. A wallet-aware fetch wrapper pays it automatically and retries once.

import { createSigner } from 'x402/types'
import { wrapFetchWithPayment } from 'x402-fetch'

// base-sepolia testnet key funded with a few cents of test USDC
// (faucet.circle.com — browser + reCAPTCHA, no scriptable path)
const signer = await createSigner('base-sepolia', process.env.PRIVATE_KEY)
const fetchWithPay = wrapFetchWithPayment(fetch, signer)

const res = await fetchWithPay('https://your-hood-api.example.com/v1/portfolio/0xYourAddress')
const portfolio = await res.json()
console.log(portfolio.totalUsd)

// mainnet: createSigner('base', process.env.PRIVATE_KEY) — pays real USDC on Base.
Settlement network is chosen by the deployment (X402_NETWORK=base or base-sepolia), not the caller. A deployment configured for Base mainnet only accepts real USDC; one configured for Base Sepolia only accepts testnet USDC. Check GET /v1/health or the 402 challenge's network field to see which.

Pricing

Free tier has no key and no quota beyond IP rate limits. Paid calls settle per-request — nothing to subscribe to, nothing to cancel.
