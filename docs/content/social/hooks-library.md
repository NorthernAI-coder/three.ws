# Hooks Library

Reusable opening lines, grouped by angle. Pair a hook with any template in [post-templates.md](post-templates.md).

**Every hook is honest by construction.** None claim engagement, user counts, or revenue. Any hook that *could* carry a number carries a `[SOURCE: …]` gate instead — fill it with a real, sourced figure or cut the clause. Never insert an invented number. `/play` is never "single-player"; IBM/watsonx is "built on watsonx.ai"; $THREE is the only coin.

### Hooks that need a real number (gate before use)

These are templated for when we *have* a sourced figure. Do not post until the gate is filled from a named source.

- "We just shipped our [SOURCE: Nth — pull from changelog/git] release. Here's what changed."
- "[SOURCE: real count from analytics] agents registered on three.ws and counting." — only if a real, citable count exists; otherwise cut entirely.
- "Forge has generated [SOURCE: real count from pipeline metrics] models." — same rule: real source or delete.

If you can't source the number, the post is stronger without it. Drop the clause and lead with the capability.
