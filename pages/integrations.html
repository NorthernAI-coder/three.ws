3D Avatar Simple

Avatar Viewer

A rotatable, animated 3D avatar — transparent background, OBS-ready, fully scriptable. Drop it on any page as a web component.

html 
<!-- Load once on your page -->
<script src="https://three.ws/embed.js" defer></script>

<!-- Drop anywhere in your HTML -->
<threews-avatar
  avatar-id="your-avatar-id"
  hide-chrome
  pose="idle"
></threews-avatar>

<!-- Optional JS control -->
<script>
  const el = document.querySelector('threews-avatar');
  await el.ready;
  el.play('wave');
</script>

Chat Agent Simple

Talking Chat Agent

A full chat interface with a live 3D character. Voice or text. Bring your AI agent to life on any site — like PumpBot.

html 
<!-- Injects a 420×600 chat agent iframe -->
<script
  src="https://three.ws/embed.js"
  data-widget="your-widget-id"
  data-type="talking-agent"
  defer
></script>

<!-- Or control the iframe with JS -->
<script src="https://three.ws/embed-sdk.js"></script>
<script>
  const agent = Agent3D.connect(iframeEl, {
    agentId: 'your-agent-id',
    onReady: () => agent.send('Hello!'),
  });
</script>

Walk Companion Simple

Walk Companion

A floating 3D character that lives in the corner of your site. It walks, gestures, and responds to your users — with one script tag.

html 
<!-- Floating corner avatar — one tag -->
<script
  src="https://three.ws/walk-embed-sdk.js"
  data-avatar="your-avatar-id"
  data-position="bottom-right"
  data-env="studio"
  defer
></script>

<!-- Control with JS -->
<script>
  const a = ThreeWalkAvatar.embed('#my-iframe');
  a.on('ready', () => {
    a.gesture('wave');
    a.say('Welcome!', 'aria');
  });
</script>

MCP Server Advanced

MCP Integration

Let any AI assistant control your 3D agents directly — render, speak, gesture, and emote from Claude, Cursor, or any MCP-compatible tool.

json 
// claude_desktop_config.json
{
  "mcpServers": {
    "threews-avatar": {
      "command": "npx",
      "args": [
        "-y",
        "@three-ws/avatar-agent-mcp"
      ],
      "env": {
        "THREEWS_API_KEY": "your-key"
      }
    }
  }
}
