Payment flow

How it works

1. The endpoint answers with a 402 Payment Required challenge (x402 v2).
2. The MetaMask Agentic CLI signs an EIP-3009 USDC transfer authorization (mm wallet sign-typed-data).
3. The request retries with the X-PAYMENT header; a facilitator verifies and settles the USDC on Base mainnet, then the API does the paid work.
