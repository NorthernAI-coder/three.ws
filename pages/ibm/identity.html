    How the firewall works

    01

    Embed with Granite

    The candidate's name + bio are embedded on watsonx.ai with granite-embedding-278m — the same vectors that power the Agent Galaxy.

    02

    Compare semantically

    Cosine similarity against every public agent surfaces look-alikes a plain name match misses. Above ~93% to another owner's agent reads as impersonation.

    03

    Screen with Guardian

    Granite Guardian classifies the identity for harm, social bias and sexual content before it can ever represent the platform.

    This is the exact gate that runs on every POST /api/agents — a blocked identity is refused at creation. Explore the rest of three.ws on IBM watsonx.
