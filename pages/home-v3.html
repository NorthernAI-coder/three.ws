07 — Install

two lines.
live.

Paste once in <head>. Use <three-d> anywhere. Works in React, Vue, Svelte, plain HTML, Webflow, Notion, Shopify — any host that allows a script tag.

// in <head>
<script src="https://three.ws/embed/v1.js" defer></script>

// anywhere
<three-d
  agent="eip155:8453:42"
  interactive autoplay
></three-d>
  • lazyBoots on intersect. Zero cost off-screen.
  • eagerAdd the attribute to force above-the-fold mount.
  • shadowIsolated CSS. No leakage in or out.
  • themeDark, light, transparent, custom.
  • idERC-8004 address or raw glTF URL.
  • size15KB loader. CDN-deduped runtime.
