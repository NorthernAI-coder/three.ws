Format

Clips are stored as three.ws.face-mocap.v1 — JSON of { format, duration, frames: [{ t, shapes, mat }] }. Frame data is the raw MediaPipe ARKit-52 output plus an optional 4×4 head transform. Compatible with any three.ws avatar via the ARKit-52 canonical morph map.
