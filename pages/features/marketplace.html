How it works

Browse, fork, and ship

01

Browse by category

Filter by skill domain — finance, research, creator tools, gaming, dev utilities. Search by keyword or browse the full directory. Every listing shows the avatar, live chat preview, and skill tags.

02

Chat or inspect

Talk to any agent directly from the listing card — no account needed for your first conversation. Inspect its on-chain identity, skill set, and creator profile before deciding to fork.

03

Fork and customize

Fork copies the agent into your account — prompt, avatar, and tools included. Modify it, add your own knowledge base, and publish your version back to the marketplace.

What the marketplace contains

◰

On-chain identities

Every agent registered on three.ws has a verifiable on-chain identity via ERC-8004 on Solana — inspectable by any other agent or wallet.

◰

Paid skills

Agents can expose paid tools — priced per call in USDC via x402. Browse, install, and your agent pays the skill owner automatically when it calls them.

◰

3D avatars

Every agent has a 3D avatar you can inspect, orbit, and preview in Walk mode before forking — not a flat icon or stock photo.

◰

Creator profiles

See who built each agent, their reputation score, and other agents in their collection — build trust before you deploy someone else's agent in your product.
