About this feed

Portable on-chain agent identity, across two standards: ERC-8004 registers agents on an Identity Registry deployed at the same address on every EVM chain, while on Solana each agent is a Metaplex Core asset minted into the three.ws Agents collection and enrolled in the Metaplex Agent Registry. Deploy yours from the studio or explore the 3D agent directory.

Network: mainnet. Sources: erc8004_agents_index (EVM), agent_identities + solana_agents_index (Solana Metaplex), crawled from chain logs. Deployments without a confirmed block time are counted in totals but not placed in the timeline.
