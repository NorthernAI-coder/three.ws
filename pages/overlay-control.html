How to wire OBS

				
    
					
  1. Click Open overlay window above — copy the URL from the new tab.
    2. 
					
  2. In OBS, add a Browser Source. Paste the URL.
    3. 
					
  3. Set width 600 px, height 800 px (or to taste). Check Refresh browser when scene becomes active.
    4. 
					
  4. Keep this tab open while you stream — every hotkey or speak action you fire here is broadcast to the OBS overlay via BroadcastChannel.
