The integration.

three.ws implements the full AWS Marketplace SaaS contract — registration, metering, entitlements, and signed SNS subscription lifecycle webhooks.

POST /api/aws-marketplace/register
Fulfillment URL. Exchanges x-amzn-marketplace-token for a stable customer identifier via ResolveCustomer.
POST /api/aws-marketplace/subscription
SNS notification webhook. Verifies AWS signature against the signing cert and processes subscribe / unsubscribe lifecycle events.
POST /api/aws-marketplace/link
Links an AWS Marketplace subscription to an authenticated three.ws account after the post-subscribe redirect.
BatchMeterUsage + GetEntitlements
Daily metering and contract entitlement checks via the AWS Marketplace Metering and Entitlement Service SDKs.
Open source. The integration lives in api/aws-marketplace/ and api/_lib/aws-marketplace.js. Review the code, fork it, run your own deployment.
